# Wazuh SIEM Home Lab

## Project Overview
This project demonstrates a SOC Analyst Level 1 home lab using Wazuh SIEM.

## Objectives
- Install Wazuh SIEM
- Monitor Linux logs
- Detect security events
- Analyze alerts
- Map detections to MITRE ATT&CK

## Lab Environment
- Ubuntu Server
- Wazuh Server
- VirtualBox
- Windows Host

## Skills Learned
- SIEM
- Linux
- Log Analysis
- Threat Detection
- Incident Investigation
